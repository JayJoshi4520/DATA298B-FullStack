
# Data Directory

This directory should contain your image data.  Organize your data into subdirectories,
where each subdirectory represents a class.

Example:

```
data/
├── cat/
│   ├── cat_image_1.jpg
│   ├── cat_image_2.jpg
│   └── ...
├── dog/
│   ├── dog_image_1.jpg
│   ├── dog_image_2.jpg
│   └── ...
└── ...
```
